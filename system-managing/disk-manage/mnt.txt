С помощью команды mount можно подключить сетевой диск, раздел жесткого диска или USB-накопитель.


узнаём информацию обо всех разделах
sudo fdisk -l

создаём директорию, в которой будут отображаться файлы с нашего внешнего носителя:
sudo mkdir /mnt/usb

монтируем:
sudo mount /dev/sdb4 /mnt/usb

Просмотр примонтированых устройств осуществляется командой:
sudo mount

Этого же результата можно достигнуть посмотрев содержимое файла /etc/mtab (в некоторых системах Linux файл называется /etc/mnt/tab)
